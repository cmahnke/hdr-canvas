Three.js HDR

HDR support for the canvas element needs the browser flag enable-experimental-web-platform-features to be enabled. For example, open chrome://flags#enable-experimental-web-platform-features in Chrome to activate it.

This model uses a UltraHDR Texture, tone mapping isn't enabled. If your browser doesn't support HDR canvases you'll just see a model with a black texture.
